# Sftp Handler
A PHP Class to handle the sftp commands.

<a name="install"></a>
## Installation

<a name="install-composer"></a>
### Via Composer

```
$ composer require citco/sftp
```

```json
{
    "require": {
        "citco/sftp": "*"
    }
}
```
